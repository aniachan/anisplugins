# Ani's Plugins

This repository is a public Dalamud plugin repository for Ani's plugins. It exists to publish plugin manifests in one place so they can be added to XIVLauncher/Dalamud as a custom repository.

Current plugins in this repository:

- [Aurum](https://github.com/aniachan/Aurum) - smart crafting profit calculator with market demand analysis.
- [Anumbra](https://github.com/aniachan/Anumbra) - Penumbra companion plugin that adds preview image support for mods.

## Repository URL

Add this repository URL to Dalamud:

```text
https://raw.githubusercontent.com/aniachan/anisplugins/main/repo.json
```

## Installing

1. Enter `/xlsettings` in game and open the Experimental tab.
2. Scroll to the Custom Plugin Repositories section.
3. Paste the repository URL above into an empty field.
4. Click the `+` button and ensure the new entry is enabled.
5. Click the Save icon in the bottom-right corner.

After that, Aurum and Anumbra will appear in the Available Plugins tab inside the Dalamud Plugin Installer.
